# User Management API (Coursera Demo Project)

This repository contains a simple User Management REST API developed as part of a Coursera course assignment.

## Project Overview
The API provides CRUD operations for managing users and demonstrates:
- RESTful API design
- GET, POST, PUT, DELETE endpoints
- Input validation
- Middleware implementation
- Debugging and optimization using Microsoft Copilot

## Technologies Used
- ASP.NET Core Web API
- C#

## Copilot Usage
Microsoft Copilot was used to generate initial API code, assist in debugging endpoint logic, implement validation checks, and create middleware for request logging.

## Note
This project is a beginner-friendly demo created for educational purposes.
